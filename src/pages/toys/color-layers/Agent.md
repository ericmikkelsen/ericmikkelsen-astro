# Color layers

This mini js app is designed to use [oxipng](https://github.com/valterkraemer/oxipng-wasm?tab=readme-ov-file) to reduce and index the colors of a png. Creating a dithered image with just a few colors. 
Each color should be treated as a separate layer with just 2 colors, transparent and the color of the layer. A basic html structure is right next to this file, and has some explainer notes. We may need to use [WebGL](https://developer.mozilla.org/en-US/docs/Web/API/WebGL_API) to generate the different layers.
